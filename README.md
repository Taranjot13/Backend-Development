## ğŸ Getting Started

To get started with this repository, you can clone it to your local machine:

```bash
git clone https://github.com/Taranjot13/Backend-Development.git
```

Navigate into a specific lecture directory to explore the code and examples. For lectures with a `package.json` file, you can install the dependencies by running:

```bash
cd class/lec-{lecture_number}
npm install
```
